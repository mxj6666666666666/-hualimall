# 启动与构建相关文件

## package.json
核心作用：告诉 npm「这个项目有哪些依赖、怎么启动」。

关键点：
- `"type": "module"`：使用 ES Module 语法（`import/export`）
- `"scripts"`：
  - `npm run dev`：本地开发
  - `npm run build`：打包生产
  - `npm run preview`：预览打包结果
- `"dependencies"`：
  - `vue`：核心框架
  - `vue-router`：页面路由
  - `pinia`：状态管理
  - `axios`：HTTP 请求

---

## vite.config.js
核心作用：配置 Vite 开发行为。

关键代码：
```js
server: {
  proxy: {
    '/api': {
      target: 'http://localhost:8080',
      rewrite: (path) => path.replace(/^\/api/, ''),
    },
  },
}
```
含义：
- 前端请求 `/api/xxx` 时，开发环境会代理到后端 `http://localhost:8080/xxx`
- 这样可以避免本地跨域问题

---

## index.html
核心作用：浏览器加载的第一个文件。

关键代码：
```html
<div id="app"></div>
<script type="module" src="./src/main.js"></script>
```
含义：
- `#app` 是 Vue 应用挂载点
- 真正应用逻辑从 `main.js` 开始

---

## README.md / package-lock.json（你现在了解即可）
- `README.md`：模板默认说明，项目里业务信息很少
- `package-lock.json`：锁定依赖版本，保证团队安装一致
