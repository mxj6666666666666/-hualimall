# 入口、根组件、路由（前端主干）

## src/main.js（应用启动入口）
关键代码：
```js
const app = createApp(App)
app.use(createPinia())
app.use(router)
app.mount('#app')
```
含义：
1. 创建 Vue 应用
2. 注册 Pinia（全局状态）
3. 注册 Router（页面切换）
4. 挂载到页面 `#app`

---

## src/App.vue（页面外壳）
关键代码：
```vue
<AppHeader />
<RouterView v-slot="{ Component }">
  <Transition name="page-fade" mode="out-in">
    <component :is="Component" />
  </Transition>
</RouterView>
```
含义：
- `AppHeader`：顶部导航固定存在
- `RouterView`：根据 URL 显示不同页面组件
- `Transition`：路由切换动画

---

## src/router/index.js（路由配置 + 权限守卫）
这个文件非常关键，它控制“谁能访问哪些页面”。

### 1) 路由表
例如：
- `/products` 商品列表
- `/cart` 购物车（需要登录）
- `/admin/*` 管理端（需要登录 + 管理员）

### 2) 动态导入
```js
component: () => import('../views/product/ProductListView.vue')
```
含义：按需加载页面，优化首屏体积。

### 3) 路由守卫 beforeEach
```js
if (to.meta.requiresAuth && !authStore.isLoggedIn) {
  return `/login?redirect=${encodeURIComponent(to.fullPath)}`
}
if (to.meta.requiresAdmin && authStore.role !== 'ADMIN') {
  return '/forbidden'
}
```
含义：
- 未登录访问受保护页 -> 跳登录并记录回跳地址
- 非管理员访问管理页 -> 跳无权限页
