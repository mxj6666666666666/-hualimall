# 页面讲解（购物车与订单）

## src/views/cart/CartView.vue（购物车）
核心能力：
1. 展示购物车项
2. 修改数量 / 勾选状态 / 删除
3. 根据勾选项计算结算信息
4. 提交订单

### 关键代码：computed 派生数据
```js
const selectedItems = computed(() => items.value.filter((item) => item.selected === 1))
const selectedCount = computed(() => selectedItems.value.length)
const selectedAmount = computed(() =>
  selectedItems.value.reduce((sum, item) => sum + Number(item.productPrice) * Number(item.quantity), 0),
)
```
含义：只要 `items` 变化，已选数量和总金额会自动更新。

## src/views/order/OrderListView.vue（订单列表）
核心能力：
- 分页拉取订单
- 显示订单号、状态、总金额
- 未支付订单可取消

关键点：
- `formatOrderStatus`：状态码转中文文本
- `cancelOrder` 成功后重新拉列表保持页面一致

---

## src/views/order/OrderDetailView.vue（订单详情）
核心能力：
- 根据 `:id` 请求订单详情
- 展示订单基础信息 + 明细列表（商品、单价、数量、小计）

---

## src/utils/format.js（格式化工具）
- `formatPrice(value)`：金额保留 2 位小数
- `formatOrderStatus(status)`：状态码映射显示文本

这类工具函数的价值：统一展示逻辑，避免每个页面各写一套。
