# HualiMall 前端项目总览（先建立全局认识）

## 1. 这是一个什么项目
- 技术栈：`Vue 3 + Vite + Vue Router + Pinia + Axios`
- 业务定位：一个商城前端，包含
  - 用户端：商品、购物车、订单
  - 管理端：商品管理、订单管理

## 2. 你可以先这样理解项目结构
```txt
client
├─ index.html                # 浏览器入口壳子（真正挂载点在 #app）
├─ package.json              # 项目依赖和 npm 脚本
├─ vite.config.js            # Vite 配置（含 /api 代理）
├─ src
│  ├─ main.js                # Vue 应用启动入口
│  ├─ App.vue                # 根组件（页面框架 + RouterView）
│  ├─ router/index.js        # 路由表 + 路由守卫
│  ├─ store/modules/auth.js  # 登录状态（token、用户信息）
│  ├─ api                    # 所有后端请求封装
│  ├─ views                  # 各个页面
│  ├─ components             # 可复用组件（如顶部导航）
│  └─ utils/format.js        # 格式化工具函数
└─ teach                     # 这次教学文档输出目录
```

## 3. 你最先要掌握的运行路径（非常重要）
1. 浏览器打开 `index.html`，看到 `<div id="app"></div>`
2. `main.js` 执行，`createApp(App).mount('#app')`
3. `App.vue` 显示头部 + `<RouterView />`
4. 路由决定显示哪个 `views/*View.vue` 页面
5. 页面通过 `api/modules/*.js` 请求后端
6. `request.js` 统一处理 token、成功/失败、401 跳转登录
