# 页面讲解（管理端与顶部导航）

## src/components/layout/AppHeader.vue（全局导航）
作用：每个页面顶部都显示。

关键逻辑：
- 根据登录状态切换“登录/注册”或“昵称/退出”
- 根据角色显示管理员入口
```vue
<RouterLink v-if="auth.role === 'ADMIN'" to="/admin/products">管理端</RouterLink>
```
- 点击退出：
```js
auth.clearAuth()
router.push('/login')
```

---

## src/views/admin/AdminProductsView.vue（管理端商品管理）
它是典型“后台表单 + 表格”页面。

核心能力：
1. 查询商品（关键词、分类、状态）
2. 新增 / 编辑商品（共用一个表单）
3. 单个删除
4. 批量勾选删除

关键思路：
- `editingId` 有值表示“编辑模式”，否则“新增模式”
- `selectedIds` 存储表格勾选项
- `toggleAll` 实现全选/取消全选

---

## src/views/admin/AdminOrdersView.vue（管理端订单管理）
核心能力：
- 拉取订单列表
- 每行选择订单状态并更新

关键代码：
```js
statusMap[item.id] = Number(item.status)
```
含义：用对象保存“订单ID -> 当前选中的状态值”，方便逐行更新。

## src/views/error/ForbiddenView.vue
作用：当路由守卫判定非管理员访问管理页时，作为统一拦截结果页。
