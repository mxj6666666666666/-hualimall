# 华丽商城后端接口总表（Spring Boot + RESTful + JWT）

## 通用约定

| 项 | 说明 |
| --- | --- |
| Base URL | 以后端启动地址为准（如 `http://localhost:8080`） |
| 鉴权方式 | `Authorization: Bearer <token>` |
| 统一返回 | `Result<T>`：`{ code, message, data }` |
| 分页返回 | `PageResult<T>`：`{ total, rows }`（通常作为 `data`） |
| 管理端权限 | `/admin/**` 除登录外都需登录，且 `role=ADMIN` |

---

## 接口清单（整合总表）

| 模块 | 方法 | 路径（含兼容别名） | 是否鉴权 | 请求参数/请求体 | data 返回结构 | 功能简述 |
| --- | --- | --- | --- | --- | --- | --- |
| 用户 | POST | `/users`、`/users/register`、`/user`、`/user/register` | 否 | Body: `User`（注册常用：`username,password,nickname`） | `String` | 用户注册 |
| 用户 | POST | `/users/sessions`、`/users/login`、`/user/sessions`、`/user/login` | 否 | Body: `LoginRequest` | `LoginResponse` | 用户登录并获取 JWT |
| 用户 | GET | `/users/me`、`/users/info`、`/user/me`、`/user/info` | 是 | Header: Bearer Token | `User` | 获取当前登录用户信息 |
| 商品 | GET | `/products/{id}` | 否 | Path: `id` | `Product` | 查询商品详情 |
| 商品 | GET | `/products`、`/products/list` | 否 | Query: `page,pageSize,categoryId,status,keyword` | `PageResult<Product>` | 商品分页列表 |
| 商品管理 | POST | `/admin/products` | 是（ADMIN） | Body: `Product` | `String` | 新增商品 |
| 商品管理 | POST | `/admin/products/batch` | 是（ADMIN） | Body: `Product[]` | `String` | 批量新增商品 |
| 商品管理 | PUT | `/admin/products/{id}` | 是（ADMIN） | Path: `id` + Body: `Product` | `String` | 更新单个商品 |
| 商品管理 | PUT | `/admin/products/batch` | 是（ADMIN） | Body: `Product[]` | `String` | 批量更新商品 |
| 商品管理 | DELETE | `/admin/products/{id}` | 是（ADMIN） | Path: `id` | `String` | 删除单个商品 |
| 商品管理 | DELETE | `/admin/products/batch` | 是（ADMIN） | Body: `Long[]`（商品ID列表） | `String` | 批量删除商品 |
| 订单 | POST | `/orders`、`/orders/create`、`/order`、`/order/create` | 是 | Body: `Order`（含 `items`） | `Order` | 创建订单 |
| 订单 | GET | `/orders`、`/orders/list`、`/order`、`/order/list` | 是 | Query: `page,pageSize` | `PageResult<Order>` | 我的订单分页列表 |
| 订单 | GET | `/orders/{id}`、`/order/{id}` | 是 | Path: `id` | `Order`（含 `items`） | 订单详情 |
| 订单 | PUT | `/orders/{id}/cancel`、`/orders/cancel/{id}`、`/order/{id}/cancel`、`/order/cancel/{id}` | 是 | Path: `id` | `String` | 取消订单 |
| 订单管理 | PUT | `/admin/orders/{id}/status`、`/admin/orders/{id}/status/update`、`/admin/order/{id}/status`、`/admin/order/{id}/status/update` | 是（ADMIN） | Path: `id` + Body: `OrderStatusUpdateRequest` | `String` | 管理端修改订单状态 |
| 购物车 | GET | `/carts`、`/carts/list`、`/cart`、`/cart/list` | 是 | 无 | `CartItem[]` | 获取当前用户购物车 |
| 购物车 | POST | `/carts`、`/carts/add`、`/cart`、`/cart/add` | 是 | Body: `CartAddRequest` | `String` | 添加商品到购物车 |
| 购物车 | PUT | `/carts/{id}`、`/cart/{id}` | 是 | Path: `id` + Body: `CartUpdateRequest` | `String` | 按购物车项ID更新 |
| 购物车 | PUT | `/carts/update`、`/cart/update` | 是 | Query: `id` + Body: `CartUpdateRequest` | `String` | 兼容式更新购物车 |
| 购物车 | DELETE | `/carts/{id}`、`/carts/remove/{id}`、`/cart/{id}`、`/cart/remove/{id}` | 是 | Path: `id` | `String` | 删除购物车项 |

## 关键请求/响应模型字段

| 模型 | 字段 |
| --- | --- |
| LoginRequest | `username`, `password` |
| LoginResponse | `token` |
| ProQueryParams | `page`(默认1), `pageSize`(默认10), `categoryId`, `status`, `keyword` |
| OrdQueryParams | `page`(默认1), `pageSize`(默认10) |
| CartAddRequest | `productId`, `quantity` |
| CartUpdateRequest | `quantity`, `selected` |
| OrderStatusUpdateRequest | `status` |
| Product | `id,name,categoryId,price,stock,imageUrl,description,status,createTime,updateTime` |
| User | `id,username,password,nickname,role,status,createTime,updateTime` |
| Order | `id,orderNo,userId,totalAmount,status,createTime,updateTime,items` |
| OrderItem | `id,orderId,productId,productName,productPrice,quantity,totalPrice` |
| CartItem | `id,userId,productId,quantity,selected,createTime,updateTime,productName,productPrice,imageUrl` |
