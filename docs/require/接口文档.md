## 接口设计（RESTful API）

### 3.1 接口清单（15个核心接口）

| 模块       | 方法   | 路径                       | 说明                      |      |
| ---------- | ------ | :------------------------- | ------------------------- | ---- |
| **用户**   | POST   | `/user/register`           | 注册                      |      |
|            | POST   | `/user/login`              | 登录（返回token）         |      |
|            | GET    | `/user/info`               | 获取个人信息（需token）   |      |
| **商品**   | GET    | /product/list`             | 商品列表（分页+分类筛选） | x    |
|            | GET    | /product/{id}`             | 商品详情                  | x    |
|            | GET    | /category/list`            | 分类列表                  |      |
| **购物车** | GET    | /cart/list`                | 购物车列表                |      |
|            | POST   | /cart/add`                 | 添加到购物车              |      |
|            | PUT    | /cart/update`              | 修改数量                  |      |
|            | DELETE | /cart/remove/{id}`         | 删除购物车项              |      |
| **订单**   | POST   | /order/create`             | 创建订单                  | x    |
|            | GET    | /order/list`               | 订单列表                  | x    |
|            | GET    | /order/{id}`               | 订单详情                  | x    |
|            | PUT    | /order/cancel/{id}`        | 取消订单                  | x    |
| **管理端** | POST   | `/admin/products`          | 添加商品                  | x    |
|            |        | `/admin/products/batch`    | 批量添加                  | x    |
|            | PUT    | `/admin/products/{id}`     | 修改商品                  | x    |
|            |        | `/admin/products/batch`    | 批量修改                  | x    |
|            | DELETE | `/admin/products/{id}`     | 删除商品                  | x    |
|            |        | `/admin/products/batch`    | 批量删除                  | x    |
|            | PUT    | `/admin/order/{id}/status` | 修改订单状态              |      |
